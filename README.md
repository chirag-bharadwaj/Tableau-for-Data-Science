# 📊 Tableau for Data Science

This repository focuses on building powerful data visualizations and dashboards using Tableau for Data Science and Business Intelligence. It helps transform raw data into meaningful insights through interactive and visually appealing dashboards.

---

## 🚀 Overview

Tableau is a leading data visualization tool used by data professionals to analyze and present data effectively. This repository provides hands-on examples and projects to help you:

* Create interactive dashboards
* Perform data analysis visually
* Tell compelling data stories
* Support data-driven decision-making

---

## 📚 Topics Covered

### 🔹 Basics

* Introduction to Tableau
* Connecting to data sources
* Data types and data roles
* Basic charts (Bar, Line, Pie)

### 🔹 Intermediate

* Filters and Sorting
* Groups and Sets
* Calculated Fields
* Parameters

### 🔹 Advanced

* Dashboards and Stories
* Level of Detail (LOD) Expressions
* Table Calculations
* Dashboard Actions

### 🔹 Data Preparation

* Data cleaning in Tableau
* Handling NULL values
* Data blending and joins
